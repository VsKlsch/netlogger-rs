# Contributing to netlogger-rs

## How to contribute

- **Bug reports** — open an issue with a description of the bug and steps to reproduce
- **Feature requests** — open an issue describing the feature and why it's useful
- **Pull requests** — fork the repository, make your changes, open a PR

## Code style

- Run `cargo fmt` before submitting
- Run `cargo clippy -- -W clippy::all` — no warnings allowed
- Add documentation for public API

## Additional ignoring for RustRover users
If you use RustRover, please take note of this https://github.com/github/gitignore/blob/main/Global/JetBrains.gitignore
